# Podstawy Kitchari
category: diet/ayurveda

## Zasady
- [ ] Baza: ryż basmati + fasolka mung (łuskana, żółta)
- [ ] Proporcja: 1:1 lub 2:1 (ryż:mung)
- [ ] Tadka: ghee + przyprawy (kmin, kurkuma, imbir, gorczyca)
- [ ] Warzywa sezonowe: marchew, cukinia, szpinak
- [ ] Gotować do bardzo miękkiej konsystencji
- [ ] Świeża kolendra na wierzch

## Notes
Kitchari to "lekarstwo" w Ajurwedzie - łatwo strawne, oczyszczające, kompletne białko. Idealne przy detoksie lub chorobie.
