# Herbaty na Młodość
category: health/drinks

## Zasady
- [ ] Rooibos + Hibiskus: antyoksydanty
- [ ] Zielona herbata jaśminowa + Mięta + cytryna: detoks, kolagen
- [ ] Pokrzywa + Skrzyp polny: włosy, paznokcie
- [ ] Rumianek + Lawenda: redukcja stresu
- [ ] Ashwagandha + Róża: adaptogeny
- [ ] Oolong: herbata czerwona na metabolizm
- [ ] Gojnik: grecki, na odporność
- [ ] Melisa: uspokajająca

## Notes
Pić regularnie. Pamiętaj o nadrzędnej zasadzie picia herbaty ze skórką mandarynki.
