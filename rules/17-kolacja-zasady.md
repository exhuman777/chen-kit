# Zasady Kolacji
category: diet/meals

## Zasady
- [ ] Lekkie warzywa jako główny składnik
- [ ] Zupy warzywne: idealne na wieczór
- [ ] Gotowane/duszone warzywa lepsze niż surowe
- [ ] Jeść minimum 2-3h przed snem
- [ ] Unikać ciężkich białek wieczorem
- [ ] Mniejsze porcje niż obiad

## Zakazy
- [ ] Unikać surówek wieczorem (trudne do strawienia)
- [ ] Unikać ciężkich deserów
- [ ] Nie jeść tuż przed snem

## Notes
Wieczorne Agni jest słabe. Ciężki posiłek = niestrawność, słaby sen.
