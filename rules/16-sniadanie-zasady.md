# Zasady Śniadania
category: diet/meals

## Zasady
- [ ] Węglowodany jako główny składnik
- [ ] Kasze: jaglana, gryczana, jęczmienna
- [ ] Owsianki: płatki owsiane, ryżowe, jęczmienne
- [ ] Ryżanki: z przyprawami rozgrzewającymi
- [ ] Ciepłe śniadanie lepsze niż zimne
- [ ] Owoce: gotowane lub duszone (łatwiej strawne)
- [ ] Przyprawy: cynamon, kardamon, imbir

## Zakazy
- [ ] Unikać zimnych płatków z zimnym mlekiem
- [ ] Unikać surowych owoców na pusty żołądek (dla niektórych)

## Notes
Poranne Agni jest słabe - ciepłe, lekkie posiłki je rozbudzają.
