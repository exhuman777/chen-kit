# Ajurwedyjski Napar CCF
category: health/ayurveda

## Zasady
- [ ] C = Cumin (kmin rzymski): 1/2 łyżeczki
- [ ] C = Coriander (nasiona kolendry): 1/2 łyżeczki
- [ ] F = Fennel (koper włoski): 1/2 łyżeczki
- [ ] Zalać wrzątkiem, zaparzać 10 minut pod przykryciem
- [ ] Pić ciepły, między posiłkami

## Notes
Wspiera trawienie, redukuje wzdęcia, balansuje wszystkie doshy. Można pić codziennie.
