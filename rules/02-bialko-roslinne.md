# Źródła Białka Roślinnego
category: diet/protein

## Zasady
- [ ] Fasolka Mung: Idealna do kitchari, oczyszcza z metali ciężkich
- [ ] Fasolka Adzuki: Wspomaga nerki (TCM), dobra na jesień/zimę
- [ ] Soczewica Czerwona: Szybka, do dań indyjskich (moczyć 1h)
- [ ] Soczewica Czarna: Na wzmocnienie nerek (moczyć całą noc)
- [ ] Tempeh: Wędzony, smażony lub naturalny
- [ ] Migdały: Bez skórki, moczone przez noc (must have ajurwedyjskie)
- [ ] Orzechy włoskie: Na pracę mózgu
- [ ] Sezam: Ziarna, tahini, gomasio
- [ ] Złote siemię lniane: Zmielone lub kleik
- [ ] Spirulina i Chlorella: Przy anemii
- [ ] Kasza gryczana: Niepalona, moczona
- [ ] Komosa ryżowa (Quinoa): Dokładnie płukać przed gotowaniem
- [ ] Tofu: Różne rodzaje (naturalne, wędzone)

## Notes
Pamiętaj o moczeniu strączków (10-12h dla fasoli) i płukaniu komosy ryżowej.
