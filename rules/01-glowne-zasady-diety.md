# Główne Zasady Diety
category: diet/core

## Zakazy
- [ ] ZAKAZ: Połączenie Cukier + Mąka + Tłuszcz (tzw. "złota trójca")
- [ ] Unikać cukru białego i cukrów prostych
- [ ] Unikać mąki pszennej (szczególnie glutenu)
- [ ] Unikać masła orzechowego (arachidowego)
- [ ] Sery: unikać lub jeść bardzo mało
- [ ] Mleko roślinne: używać tylko warunkowo (rzadko)

## Zasady
- [ ] Śniadanie: Węglowodany (kasze, owsianki, ryżanki)
- [ ] Obiad: Warzywa + Białko wegańskie (strączki, tofu, tempeh)
- [ ] Kolacja: Lekkie warzywa
- [ ] Miód: używać tylko warunkowo (rzadko)
- [ ] Pić herbatę ze skórką z mandarynki

## Notes
Te zasady są nadrzędne wobec składników w przepisach. Jeśli przepis zawiera masło orzechowe lub mąkę pszenną, szukaj zamienników.
