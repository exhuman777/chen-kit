# Reset Układu Nerwowego
category: health/wellness

## Zasady
- [ ] Oddech 4-7-8: Wdech (4s), Zatrzymanie (7s), Wydech (8s) - powtórz 3-5x
- [ ] Dłoń na sercu i brzuchu podczas oddychania
- [ ] Rozciąganie i "wytrząsanie" napięcia z ciała
- [ ] Chodzenie boso (grounding)
- [ ] Reset temperaturą: zimna woda na twarz lub naprzemienny prysznic
- [ ] Ciepła kąpiel z solą magnezową
- [ ] 2 minuty ciszy lub dźwięki natury (ptaki)
- [ ] Dziennik: pytanie "Czego teraz potrzebuję?"
- [ ] Samowspółczucie: "Nie muszę się spieszyć"

## Notes
Stosować w momentach lęku, chaosu w głowie lub napięcia ciała.
