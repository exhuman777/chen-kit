# Moczenie Roślin Strączkowych
category: diet/preparation

## Zasady
- [ ] Fasola (wszystkie rodzaje): 10-12 godzin
- [ ] Ciecierzyca: 8-12 godzin
- [ ] Soczewica czerwona: 1-2 godziny (opcjonalnie)
- [ ] Soczewica zielona/brązowa: 4-6 godzin
- [ ] Fasolka mung: 4-8 godzin
- [ ] Kasza gryczana niepalona: 2-4 godziny
- [ ] Quinoa: płukać dokładnie (saponiny!)
- [ ] Migdały: całą noc, zdjąć skórkę

## Notes
Moczenie redukuje kwas fitynowy, ułatwia trawienie, skraca czas gotowania.
