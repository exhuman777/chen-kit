## Źródła Białka (Wegańskie)
- [ ] **Fasolka Mung:** Oczyszcza z metali ciężkich, lekkostrawna (baza do kitchari).
- [ ] **Soczewica Czarna:** Wzmacnia nerki (moczyć całą noc).
- [ ] **Soczewica Czerwona:** Szybka, do dań indyjskich (moczyć 1h).
- [ ] **Tempeh/Tofu:** Wędzone, naturalne.
- [ ] **Orzechy:** Włoskie ("mózgi"), Migdały (bez skórki, moczone).
- [ ] **Złote siemię lniane:** Zmielone lub kleik (na jelita).
- [ ] **Spirulina/Chlorella:** Przy anemii.

## Bulion Mocy (Wzmacnianie Zupy)
Aby zupa stała się lekarstwem, dodaj:
- [ ] Domowy, długo gotowany bulion.
- [ ] Świeży imbir (kawałek).
- [ ] Owoce Goji (łyżeczka).
- [ ] Czerwone daktyle (Jujubes) - 2 sztuki bez pestek (wrzucić na początku).
- [ ] Czarny sezam.

## Zioła Anti-Aging (Herbaty)
- [ ] **Rooibos + Hibiskus:** Antyoksydanty.
- [ ] **Zielona herbata + Mięta + Cytryna:** Detoks, kolagen (natural glow).
- [ ] **Pokrzywa + Skrzyp:** Włosy, paznokcie.
- [ ] **Rumianek + Lawenda:** Redukcja stresu (mniej zmarszczek).
- [ ] **Ashwagandha + Róża:** Adaptogeny dla cery.

## Notes
Pamiętaj o moczeniu strączków (Fasole: 10-12h, Soczewice: min. 1h lub noc) dla lepszego trawienia.
