# Zdrowe Zamienniki
category: diet/substitutes

## Zasady
- [ ] Masło orzechowe → Tahini lub masło ze słonecznika
- [ ] Mąka pszenna → Mąka z ciecierzycy, ryżowa, gryczana
- [ ] Cukier biały → Daktyle, syrop klonowy, ksylitol
- [ ] Mleko krowie → Mleko kokosowe, migdałowe, owsiane
- [ ] Ser żółty → Płatki drożdżowe, wegański ser
- [ ] Jajka → Siemię lniane + woda, chia + woda, aquafaba
- [ ] Śmietana → Mleczko kokosowe (gęsta część)
- [ ] Majonez → Wegański majonez lub tahini + cytryna
- [ ] Masło → Ghee, masło wegańskie, olej kokosowy

## Notes
Zawsze czytaj etykiety "wegańskich" produktów - mogą zawierać cukier i inne niechciane składniki.
